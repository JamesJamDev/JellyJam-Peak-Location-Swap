**MOD INFO**
Name: JellyJamPeakDamageSwapMod
Version: 1.0.0
Game: PEAK

**MOD FEATURES**
- Players with the mod will **swap positions with another modded player when taking damage**
- Each player has an individual **~2 second cooldown** before they can trigger another swap
- Swaps only occur between players who have the mod installed

**CAUSES A SWAP**
- Injury  
- Poison  
- Cold  
- Heat  
- Thorns  
- Drowsy  
- Curse  

**DOES NOT CAUSE A SWAP**
- Weight *(the game counts this as damage, but it is excluded)*  
- Hunger  

**Installation**
1. Install and download BepInEx
2. Drag and drop the BepInEx file contents into the game
3. Launch the game, this will generate the required files
4. Close the game, and put the DLL file inside of the plugins folder (BepInEx -> plugins)
